# Geoguessr_Hackathon

Pour lancer le projet 

cloner le github 

Deux facon de lancer l'app 

Soit faire 

Dans le dossier geoguessr
`npm i`
`npm run dev`

Dans le dossier Api

`go run main.go`

Ou sinon 

Lancer les Dockerfile qui se situe dans les dossier geoguessr et Api 
les command pour les lancer sont dedans
